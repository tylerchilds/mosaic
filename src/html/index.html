Why another grid system?

Simple. All the other ones did the same thing which didn't accomplish the things they could have aspired to accomplish.

That's arrogant, what does it do?

Simple. This system differs from other grids in that it was built for vertical alignment and for mixed fluid and fixed columns. Yeah, fixed and fluid columns in the same row. Oh yeah, Vertical alignment just by adding middle as a class to a cell.

Alignment Examples

Cells can be horizontally aligned by adding the class "left", "center", or "right" on to the row

Cells can be vertically aligned by adding "top", "middle", or "bottom" to the cell. The horizontal alignment classes can also be applied to cells.
